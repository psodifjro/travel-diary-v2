# Дневник путешественника

Веб-приложение «Дневник путешествий», разработанное в рамках производственной практики.  
Пользователи могут создавать аккаунт, публиковать записи о путешествиях и просматривать записи других пользователей.
Приложение выполнено в современном дизайне с боковой навигацией, лентой в стиле журнала
и страницей путешествия в формате "reading view" .

## Функционал (соответствие требованиям задания)

 Создан web-сайт (Flask + HTML/CSS/JS)  
 Создан пользователь системы (регистрация/вход)  
 Реализована функция записи своего путешествия (создание Trip)  
 Реализован просмотр путешествий других пользователей (страница Explore)  

### Реализованные 3+ функции о путешествии
1. Местоположение — страна, город + координаты lat/lng  
2. Изображение мест — ссылка на изображение (image_url)  
3. Стоимость путешествия — бюджет в рублях (budget)  
4. Оценки (дополнительно) — безопасность/транспорт/людность/зелень (1..5)

## Технологии

Backend
- Python 3
- Flask
- Flask-Login
- Flask-SQLAlchemy
- SQLite

Frontend
- HTML5 / Jinja2
- CSS3 (современный стиль, glass UI)
- JavaScript
- tsParticles (анимация фона)
